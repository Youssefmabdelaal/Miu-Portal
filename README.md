# Smart University Course Registration System

A complete web application for university course registration with student and admin features.

## Project Structure

- `/css` - External CSS files for styling
- `/js` - External JavaScript files for functionality
- `/pages` - HTML pages for different sections
- `/assets` - Images and other assets
- `/backend` - Backend code (to be implemented)

## Features

- Student registration and login
- Course viewing and registration
- Admin course management
- Real-time updates
- Responsive design

## Getting Started

Open `index.html` in a web browser to start the application.
