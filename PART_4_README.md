Smart University Course Registration System
A complete web application for university course registration with student and admin features

Project Structure
/css - External CSS files for styling (admin-panel.css)

/js - External JavaScript files for functionality (admin-panel.js)

/pages - HTML pages for different sections (admin-panel.html)

/assets - Images and other assets

/backend - Backend code (to be implemented)

Part 4 Features (Admin Panel)
Course Management: Full CRUD operations allowing admins to add, edit, delete, and view courses.

Real-Time Search & Filter: Instantly filter the course table by course name, code, or instructor.

Client-Side Validation: Strict form validation enforcing correct formats for course codes (e.g., CS101), seat limits (1-500), and preventing duplicate entries.

Data Persistence: Utilizes browser localStorage to save course additions and edits across sessions.

Responsive UI: Mobile-first dashboard featuring dynamic statistic updates and protective delete-confirmation modals.

Getting Started
1) Navigate to the /pages directory.

2) Open admin-panel.html in any modern web browser to start the application.

3) The table will initialize with a set of sample courses to demonstrate functionality.
